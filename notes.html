What I had to learn?

I had to learn how to use p5.js in order to draw onto a canvas element. I learned this through analyzing code snippets from "The Nature of Code" but also through heavy experimentation with my own code. I also had to learn how to integrate canvas with p5.js properly, and how p5.js would allow me to display what I wanted.

Above and Beyond

The most I went beyond for was the autonomous agent (vehicle) code. This uses linear algebraic concepts in order to map out agents throughout the canvas that behave realistically. The code is similar to things I've worked on in other classes, however the translation into Javascript is not perfect and was pretty difficult to completely figure out. Besides code, figuring out canvas and how p5.js interacted with it was another core portion of the project not covered by the regular class materials.
